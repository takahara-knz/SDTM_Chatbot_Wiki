# 申請・承認プロセス

規制当局への申請・承認に関する規制要件・手続きの領域。

## 主要トピック

### 米国（FDA）
- IND（Investigational New Drug Application）
- NDA（New Drug Application）
- BLA（Biologics License Application）
- Fast Track / Breakthrough Therapy / Accelerated Approval / Priority Review
- Rolling Review

### 欧州（EMA）
- CTA（Clinical Trial Authorization）
- MAA（Marketing Authorisation Application）
- PRIME（Priority Medicines）スキーム
- 条件付き承認

### 日本（PMDA）
- 治験届
- J-NDA（新薬承認申請）
- 希少疾病用医薬品（オーファン指定）
- 先駆的医薬品指定制度
- 条件付き早期承認制度

### 国際共通
- CTD（Common Technical Document）— ICH M4
- eCTD（電子CTD）
- ICH M8 — eCTD仕様
- 小児開発計画（PIP/PSP）

### 電子データ提出
- FDA: CDISC準拠（SDTM/ADaM）必須
- PMDA: CDISC準拠ガイドライン

- [[電磁的記録]] — 真正性・見読性・保存性の3要件
- [[電子署名-規制要件]] — 電子署名の4要件（氏名・日時・意味・リンク）
