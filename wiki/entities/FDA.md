# FDA — 米国食品医薬品局

## 概要

FDAは米国の医薬品・医療機器・食品等の規制機関。医薬品の臨床試験はCenter for Drug Evaluation and Research（CDER）またはCenter for Biologics Evaluation and Research（CBER）が所管する。

## 関連センター

| センター | 所管 |
|---------|------|
| CDER | 低分子医薬品、一部バイオ |
| CBER | 生物製剤（ワクチン、血液製剤等） |
| CDRH | 医療機器 |
| CFSAN | 食品・栄養補助食品 |

## 主要規制

| 規制 | 内容 |
|------|------|
| 21 CFR Part 50 | インフォームドコンセント |
| 21 CFR Part 56 | IRB |
| 21 CFR Part 312 | IND規制 |
| 21 CFR Part 314 | NDA規制 |
| 21 CFR Part 601 | BLA規制 |

## 優先審査制度

| 制度 | 条件 | メリット |
|------|------|---------|
| Fast Track | 重篤疾患・unmet need | ローリングレビュー |
| Breakthrough Therapy | 既存治療より実質的改善の予備的証拠 | 集中的指導、ローリングレビュー |
| Accelerated Approval | 代替エンドポイント | 市販後試験義務 |
| Priority Review | 重篤疾患・改善 | 審査期間6ヶ月 |
| RMAT | 再生医療 | Breakthrough同等 |

## 主要ガイダンス文書カテゴリ

- 臨床試験デザイン
- 適応デザイン（2019）
- 分散型臨床試験（DCT, 2023）
- エンドポイント（疾患別多数）
- 統計・解析
- 安全性報告
- CDISC準拠データ提出
