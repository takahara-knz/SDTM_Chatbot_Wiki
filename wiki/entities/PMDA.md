# PMDA — 医薬品医療機器総合機構

## 概要

PMDAは日本の厚生労働省（MHLW）の下で、医薬品・医療機器の審査・安全対策・健康被害救済を担う独立行政法人。臨床試験の相談・審査を実施する。

## 主要業務

| 業務 | 内容 |
|------|------|
| 承認審査 | 新薬・医療機器・再生医療等製品の審査 |
| 安全対策 | 市販後安全性監視、副作用報告収集 |
| 健康被害救済 | 副作用救済給付 |
| 相談業務 | 事前面談、対面助言 |

## 規制体系

| 法令 | 内容 |
|------|------|
| 薬機法（医薬品医療機器等法） | 基本法 |
| 薬事法施行規則 | 治験届、申請手続き |
| GCP省令 | 臨床試験実施基準（ICH E6に準拠） |
| 副作用報告省令 | 安全性報告義務 |

## 治験・申請プロセス

| ステップ | 内容 |
|---------|------|
| 治験届 | Phase I〜III開始前に提出（30日待機） |
| 対面助言 | 申請前相談（Phase II終了後が一般的） |
| J-NDA（新薬承認申請） | CTD形式で提出 |
| 優先審査 | 重篤疾患・医療ニーズ高い品目 |

## 優先制度

| 制度 | 説明 |
|------|------|
| 先駆的医薬品指定 | 革新的医薬品・早期承認目標（6ヶ月） |
| 条件付き早期承認制度 | 小規模集団・代替エンドポイント活用 |
| 希少疾病用医薬品 | オーファン指定（優先審査・試験費用助成） |
| 特定用途医薬品 | 小児・感染症等 |
