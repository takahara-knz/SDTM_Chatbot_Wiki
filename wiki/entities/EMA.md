# EMA — 欧州医薬品庁

## 概要

EMAはEU加盟国の医薬品規制を調整・評価する機関。本部はアムステルダム（2019年ロンドンより移転）。医薬品の中央承認審査（Centralised Procedure）を担当する。

## 主要委員会

| 委員会 | 役割 |
|---------|------|
| CHMP | ヒト用医薬品委員会（主要審査体） |
| PRAC | 薬剤警戒リスク評価委員会 |
| CAT | 先進治療薬委員会 |
| COMP | 希少疾病用医薬品委員会 |
| PDCO | 小児薬委員会 |

## 承認手続き

| 手続き | 概要 |
|---------|------|
| Centralised Procedure | EMA中央審査（バイオ、希少疾病等は義務） |
| Decentralised Procedure | 複数国同時申請 |
| Mutual Recognition | 1国承認を他国に相互承認 |
| National Procedure | 単一国申請 |

## EU臨床試験規則（CTR）

- Regulation (EU) No 536/2014（EU CTR）— 2022年1月施行
- CTIS（Clinical Trials Information System）による一元化申請
- 旧EU臨床試験指令（2001/20/EC）から移行

## 優先制度

| 制度 | 説明 |
|------|------|
| PRIME | Priority Medicines（革新的医薬品の早期対話） |
| Conditional MA | 条件付き承認（市販後データ収集義務） |
| Exceptional Circumstances | 例外的状況承認 |
| Accelerated Assessment | 審査期間150日（重大な公衆衛生上の利益） |

## 主要ガイドライン（CHMP）

- Statistical methodology guidelines
- Missing data in confirmatory trials (2010)
- Adaptive Designs guideline
- Biostatistical methodology

## 薬剤警戒

- GVP（Good Pharmacovigilance Practices）Moduleシリーズ
- EudraVigilance — EU有害事象報告データベース
- RMP（Risk Management Plan）義務
