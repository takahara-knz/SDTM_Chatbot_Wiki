# LLM-Wikiの使い方

## 1. ファイルのアップロード

- .rawフォルダに移動します
- 右上の[Add file]をクリックします
- [Upload files]を選んで、アップロードしたいファイルを放り込みます。
  対応しているファイルは、.pdf か text ファイル（.md など）になります。
  Word ファイル（.docx）は対応していないため、一度 PDF に変換してからアップロードしてください。
- 画面左下の[Commit changes]ボタンを押して、ファイル登録を確定します。

アップロードされたファイルをもとに、Hamanoにて、定期的にWikiの更新を行います。

## 2. Wikiフォルダの見方

Wikiの中身は、以下のいずれかの方法で確認できます。

- **そのままGitHub上で見る**
  Wikiフォルダをブラウザで開けば、それだけでも内容は確認できます。ちょっとした確認程度であれば、これが一番手軽です。

- **手元にダウンロードして見る**
  ページ内のリンクをたどりながらじっくり見たい場合は、Wikiフォルダを手元にダウンロードするのがおすすめです。GitHub自体には「特定のフォルダだけ」をダウンロードする機能はありませんが、以下の外部ツールを使うと、Wikiフォルダだけをまとめてダウンロードできます。

  1. [download-directory.github.io](https://download-directory.github.io/) を開きます
  2. Wikiフォルダを開いたときのGitHubのURLをコピーして、ツールの入力欄に貼り付けます
  3. Enterを押すと、そのフォルダの中身だけがZIPでダウンロードされます

  ※このツールはGitHub公式のものではなく、有志が公開しているWebツールです。サインインやトークンの入力は不要です。

## 3. Obsidianで見る

Wikiフォルダはリンクで相互に繋がっているため、Obsidianというアプリを使うと、そのリンクをたどりながら手元で見やすくなります。

### Obsidianの入手・インストール

1. [obsidian.md](https://obsidian.md/) にアクセスします
2. お使いのOS（Windows / Mac）向けのインストーラーをダウンロードします
3. インストーラーを実行し、案内に沿ってインストールします

### Vaultの設定

1. Obsidianを起動し、「Open folder as vault」を選びます
2. 「2. Wikiフォルダの見方」でダウンロードしたWikiフォルダを選択します
3. これでそのフォルダが Vault として開かれ、フォルダ内のリンクをたどりながら閲覧できるようになります
